



# 🌍 City Explorer | مستكشف المدن

A simple web application that helps users discover nearby places in any city using Geoapify API.
تطبيق ويب بسيط يساعد المستخدم على استكشاف الأماكن القريبة في أي مدينة باستخدام Geoapify API.

## 🚀 About the Project | نبذة عن المشروع

This project allows users to search for a city and display nearby places such as restaurants, entertainment spots, and commercial buildings.
يعرض المشروع أماكن قريبة من المدينة المدخلة مثل المطاعم وأماكن الترفيه والمباني التجارية.

---

## ✨ Features | المميزات

* 🔍 Search any city easily | البحث عن أي مدينة بسهولة
* 📍 Show nearby places | عرض الأماكن القريبة
* 🍽️ Restaurants, entertainment, and commercial categories | فئات المطاعم والترفيه والمباني التجارية
* 🗺️ Google Maps direct link | رابط مباشر إلى Google Maps
* 📱 Responsive design for all devices | تصميم متجاوب مع جميع الأجهزة

---

## 🛠️ Technologies | التقنيات المستخدمة
